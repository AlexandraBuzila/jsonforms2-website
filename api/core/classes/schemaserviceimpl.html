Private selfContainSchema

  • selfContainSchema(schema: JsonSchema, outerSchema: JsonSchema, outerReference: string, includedDefs?: string[]): void
    • Defined in legacy/schema.service.impl.ts:273

    Makes the given JsonSchema self-contained. This means all referenced definitions are contained in the schema's definitions block and references equal to outerReference are set to root ('#').

    Parameters

    • schema: JsonSchema

      The current schema to make self contained

    • outerSchema: JsonSchema

      The root schema to which missing definitions are added

    • outerReference: string

      The reference which is considered to be self ('#')

    • Default value includedDefs: string[] = ['#']

      The list of definitions which were already added to the outer schema

    Returns void
